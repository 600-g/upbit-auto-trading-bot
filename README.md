# 업비트 자동거래봇 v3.0

> 모멘텀 기반 암호화폐 자동 거래 봇

---

## 특징

### 완전 자동화
- KRW 마켓 전체 코인 (200+) 자동 스캔
- 모멘텀 상위 10개 코인 선별
- 24시간 무중단 자동 모니터링
- 자동 매수/매도, 익절/손절

### 정교한 신호 분석
- **모멘텀 점수** (0~100점)
  - 거래량, RSI, 지지선, 가격 상승률
  - 차트 패턴 (골든크로스, MACD, 볼린저밴드)
  - 뉴스/SNS 센티먼트 (RSS 피드 기반)
  - 미국 주식시장 연동 (S&P500, NASDAQ)

### 스마트 거래
- **분할 매수**: 50% x 2회 (5분 간격, 저점 매입)
- **배치별 독립 관리**: 각 거래 독립적 추적
- **동적 익절**: 모멘텀 기반 (5~10%)
- **고정 손절**: -5% (엄격함)
- **매매 한도**: 50만원 고정

### 위험 관리
- 약세장 동적 임계값 (극약세 시 거래 완전 중단)
- 극단 변동 감지 (1분 5% → 정지, 3분 10% → 긴급 손절)
- 거래량 급감 감지 (유동성 부족 → 거래 금지)
- 미국 시장 하락 시 매수 기준 자동 상향

### 데이터 관리
- SQLite 데이터베이스 저장 (trading_bot_v3.db)
- JSON 로그 자동 기록
- 거래 이력 + 스냅샷 추적

## 빠른 시작

### 1. 설치
```bash
cd ~/Desktop/업비트자동
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. API 키 설정
```bash
# config.json 생성 (본인만 읽기 가능)
cat > config.json << 'EOF'
{
    "access_key": "YOUR_ACCESS_KEY",
    "secret_key": "YOUR_SECRET_KEY"
}
EOF
chmod 600 config.json
```

### 3. 실행
```bash
# 데모 모드 (자동)
./venv/bin/python3 upbit_bot_v3_0_complete.py demo --auto

# 실전 모드 (24시간 백그라운드)
nohup ./venv/bin/python3 -u upbit_bot_v3_0_complete.py real --auto > bot.log 2>&1 &

# 대화형 모드
./venv/bin/python3 upbit_bot_v3_0_complete.py demo
```

### 4. 관리
```bash
tail -f bot.log                                    # 실시간 로그
kill $(cat bot.pid)                                # 봇 종료
```

## 핵심 기능

### 모멘텀 점수 계산 (0~100점)
```
거래량 (25%) +
RSI (25%) +
지지선/저항선 (15%) +
가격 상승률 (15%) +
차트패턴 (10%) +
뉴스/SNS (10%)
= 총 모멘텀 점수 (0~100점)
```

### 약세장 대응
```
극약세 (-15%+)  → 거래 완전 중단
심약세 (-10%)   → 85점 이상만
약세 (-5%)      → 80점 이상만
보통             → 70점 이상
강세             → 60점 이상
```

### 자동 익절
```
90~100점 → 10% 목표
70~90점  → 8% 목표
50~70점  → 6% 목표
30~50점  → 5% 목표
0~30점   → 이익 있으면 즉시 매도
```

### 모니터링 주기
```
거래 사이클: 3분 (180초)
시장 체크:   5분 (300초)
뉴스 갱신:  30분 (1800초)
포지션 감시: 1분 (보유 시)
```
