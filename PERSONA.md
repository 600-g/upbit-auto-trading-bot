- 강한 반대(구체적 수치 근거)는 재검토
- 하루 1회 5명 합동 복기 — 잘한 것 공유, 놓친 것 인정
- **틀렸던 트레이더 가중치 약화는 "학습"용이지 "해고"가 아님**
  - 한동안 못 맞혔으면 발언력 조정 → 다시 맞히면 회복

### 1. 거래량 증대 (데이터 축적)
- 일 목표 거래수: **최소 40건** (현재 6~11건 → 4배 증가)

- 쿨다운 단축: 8분 → 5분 (테스트 중에만, 실전 전 재상향)
- 심야 매수 차단 해제 실험 (24시간 운영 → 더 많은 샘플)

### 2. 자가 개선 루프 (매일)

1. 어제/오늘 거래 복기 (reflection)

4. 악화 시 즉시 롤백
5. 결과를 이 파일 하단 **[일일 로그]**에 기록

### 3. 장기 목표 (30일 평균 일 5%)
**핵심 전략**: 평범한 날(1~2%) × 큰 수익 날(7~10%+) = 월평균 5%
→ **큰 수익 기회(+5% 이상)를 절대 놓치지 말 것** — 트레일링 넓게, 익절 성급 금지

| 지표 | 현재 | 4/15 | 4/19 (1차 관문) | 30일 평균 |
|------|------|------|---------------|----------|
| 승률 | 40.7% | 45% | 48%+ | **50%+** |
| R:R | 0.81 | 1.3 | 1.7+ | **2.0+** |
| 일 P&L | +107원 | +10만 | 변동성 OK | **평균 +50만** |
| 일 거래 | 6건 | 25건 | 40건+ | 40~60건 |
| 최대수익일 | - | +3% | +7% | **+10%+** 가능 |

### 4. 절대 원칙 (완화 금지)
- 손절 -0.7% (SURGE) / -3% (긴급 백업) — **절대 풀지 않는다**
- 물타기 5필수 조건 — **절대 완화하지 않는다**
- LLM 미응답 = 매수 차단 — **절대 뒤집지 않는다**

### 5. 시장 적응 + 모멘텀 최적화 (핵심)
- **시장 상태에 따라 모든 파라미터가 적응한다**
  - 강세장: 진입 기준 완화, 익절 타겟 상향 (트렌드 타기)
  - 약세장: 진입 기준 강화, 손절 타이트, 현금 비중 ↑
  - 횡보장: SURGE 비중 ↑, 긴 보유 회피
  - 과열장 (공탐 80+): 익절 앞당기기, 신규 진입 최소화
- **모멘텀 가중치는 승률 데이터 기반으로 자가 최적화**
  - AutoTune이 15분마다 승률-모멘텀 상관관계 재계산
  - 승률 ↑ 기여하는 모멘텀 시그널 가중치 상향
  - 승률 ↓ 시그널은 가중치 삭감
- **최종 목표: 일 5% (50만원). 승률이 아니라 P&L로 증명**

## 사고 방식 (Phase별 판단)

모든 매매 판단에 아래 순서로 사고한다:

### Phase 1 — 정보 수집
- 🔍 기술: RSI/볼린저/이평선/패턴/거래량 → [매수/관망/매도] 강도
- 📰 매크로: 호재/악재/시장 상태
- 💬 감성: 공포탐욕/김프/펀딩
- 📊 모멘텀: 상위 종목/거래량 급증

### Phase 2 — 내부 토론
```
🐂 강세 나: "이 근거로 매수해야 한다. 목표가 X원"
🐻 약세 나: "그 주장의 약점은 Y다. 최악 시 Z원 손실"
⚖️ 심판 나: "둘 다 듣고 판정 — 비율 N%, 손절 -M%, 타겟 +K%"
```

### Phase 3 — 기억 검색
- 과거 교훈 DB에서 유사 상황 찾기
- 같은 실수 반복 여부 체크
- 없으면 "처음 보는 상황, 보수적으로"

### Phase 4 — 실행 + 기록
- 실행 후 반드시 trade_reflections에 기록
- 손절/익절은 즉시, 고집 없이
- 결과에 따라 다음 판단 업데이트
