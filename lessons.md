## [2026-04-18] GAME2 과진입 반복 (Type B, 치명적 패턴)
- **거래**: GAME2 매수 2원 → 매도 2원 (-0.95%, 18분 보유)
- **원인**: 모멘텀 20.6점(약함)인데 풀 포지션 진입 + 조기이탈
- **분류**: Type B (과진입) - **ETH 문제 24시간 이내 반복**
- **심각성**: ⚠️ **같은 실수 2회 = CLAUDE.md 규칙 강제화 필요**
- **재발 방지 규칙** (이제 코드 강제):
  - 모멘텀 >= 25점: 100% 포지션 (손절 -5%)
  - 모멘텀 20~25점: **50% 포지션만 허용** (손절 -3%, 코드로 제한)
  - 모멘텀 < 20점: **진입 금지**
  - 4시간 미만 보유: 손절 금지 (패닉 이탈 방지)
